Elasticsearch cluster availability has been affected

One of our Elasticsearch cluster's availability had been affected for a few hours on March 13, 2017. On investigating, the cause has been a memory leak in environments that use Docker v1.9 and Linux Kernel 4.3 environments. https://github.com/docker/docker/issues/18295 is an open issue.

Update 2017-03-13 17:42:42 UTC

A good follow-up step should be to upgrade the host and docker environments to the most stable recent release.

User login issue

A section of users logging in with Github are affected by a 500 error when they log in, the issue was reported on Feb 13th.
